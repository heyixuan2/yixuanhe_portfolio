Overview

As part of the HRGC "Re-Skill and Re-Invent" initiative, I independently designed, developed, and delivered a structured digital transformation program spanning two phases, with the guidance of my mentor Kieso Jan. The program was part of a broader organizational strategy to shut down legacy HR reporting systems and migrate to a centralized Power BI Service cloud environment with semantic models, row-level security, and scheduled data refresh.

Phase I, the MBGC Power BI Insights Bootcamp, ran an 8-session self-paced curriculum covering the full Power BI Desktop-to-Service pipeline: data importing, table relationships, DAX measures, Power Query transformations, report design, row-level security, and Power BI Service publishing with semantic model management. Each week included approximately 30-minute video lectures, cumulative practice sets with 10 to 15 tasks using real HR datasets, lecture quizzes, and practice set quizzes administered via Testportal — with weekly performance scores reported to management.

Phase II, the MBGC M365 Excellence Lab, extended the program with scenario-based M365 training: cloud file management via SharePoint and OneDrive, workflow automation with Power Automate, project planning with Planner and Whiteboard, and multimedia content production with Clipchamp. The program was recognized by HRGC executive leadership and presented directly to Mercedes-Benz Board Member Sabine Kohleisen.

Phase 1

MBGC Power BI Insights Bootcamp — 8 Sessions, 26 Videos

A progressive, self-paced curriculum covering the complete Power BI workflow — from data importing and modeling through advanced DAX expressions, Power Query transformations, row-level security, and Power BI Service deployment with semantic models. Each session includes approximately 30-minute self-guided video lectures, cumulative practice sets of 10 to 15 tasks per session built on real HR datasets, and two quizzes per week — one for lecture content and one for the practice set — with scores reported to management. The program included weekly optional walk-in workshops, a Microsoft Teams community with Q&A support, and a competitive prize system with weekly and overall performance awards.
