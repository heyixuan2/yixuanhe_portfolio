Drivers of Flight Disruptions at DCA Airport

Fall 2025 — Data Analytics

A data analytics project examining factors that cause flight delays and cancellations at Ronald Reagan Washington National Airport (DCA). The project integrates 2023–2024 FAA flight operational data with METAR weather data and holiday calendars to identify disruption patterns and develop actionable insights.

R Power BI FAA Data METAR Weather Hypothesis Testing Data Integration
Power BI Dashboard — Add screenshot or embed here Power BI iframe or screenshot (1200 x 675px)

Data Sources

  • FAA flight operational data: monthly CSV files covering 2023–2024, filtered for DCA origin/destination flights
  • METAR weather data: 15+ hourly variables including temperature, wind speed, precipitation, visibility, pressure, humidity, and sky conditions
  • Holiday calendar dataset for seasonal and event-based analysis
Delay trend chart
Weather correlation analysis

Methodology & Approach

  • Data integration pipeline merging FAA flight records with hourly METAR weather observations by timestamp and location
  • Feature engineering: delay flag indicators, cancellation reason mapping, delay breakdown by cause (carrier, weather, NAS, security, late aircraft)
  • Statistical hypothesis testing (t-tests) to validate significance of weather and temporal factors on delay probability
  • Power BI interactive dashboards showing weekly delay trends, cancellation rates, and cause-breakdown analysis
Delay cause breakdown
Statistical test results

Key Outcomes

  • Identified weather variables with strongest correlation to flight delays — visibility and wind speed emerged as primary disruption drivers
  • Built comprehensive data cleaning pipeline documenting all transformations for reproducibility
  • Power BI dashboards providing actionable insights for stakeholders on delay patterns by carrier, season, and weather condition
  • Final report synthesizing quantitative findings with practical recommendations for flight disruption mitigation
