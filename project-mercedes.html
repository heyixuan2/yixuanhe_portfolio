04 / PROFESSIONAL — MERCEDES-BENZ

AI-Powered CV Screening & HR Analytics Platform

Summer 2025 — People Analytics & AI Internship, Beijing

Architected an end-to-end AI-powered CV screening prototype that automated the candidate evaluation pipeline — from data extraction to AI-driven scoring. Integrated enterprise-grade APIs across major cloud platforms and built real-time HR analytics dashboards, reducing manual screening time by 90%.

The Challenge

Mercedes-Benz HR teams across Greater China processed thousands of candidate applications manually, resulting in significant time investment and inconsistent evaluation criteria. The team needed an automated, AI-driven solution that could integrate with existing HR systems while maintaining enterprise-grade security and compliance.

Solution Architecture

  • Built a fully automated pipeline using Dify AI workflows for intelligent document processing and candidate scoring
  • Python automation layer for data extraction, transformation, and orchestration between AI models and data stores
  • Enterprise-grade API integrations across Google Cloud, OpenAI, Microsoft Azure, and Dify platforms
  • Automated OAuth2 authentication for secure, real-time data exchange across all connected services
  • Power BI dashboards providing HR leadership with real-time, AI-enhanced talent insights and scoring metrics
