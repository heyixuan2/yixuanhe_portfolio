01 / DSAN 5000 — GEORGETOWN UNIVERSITY

Police Incident Patterns & Response Time Modeling

Fall 2025 — Data Science & Analytics

A comprehensive analysis of police dispatch patterns and emergency response times in Montgomery County, Maryland. This project investigates what factors drive dispatch-to-arrival times, how response times vary by time, location, and call type, and builds predictive models for call priorities and disposition outcomes.

R Quarto Clustering Regression Classification EDA Data Storytelling
Project overview visualization — Add screenshot here Recommended: 1200 x 675px

Data Source

Real-world police dispatch data from Montgomery County's open data API, covering incident records with call types, priority levels, timestamps, geographic coordinates, and disposition outcomes.

EDA visualization
Spatial clustering map

Methodology & Approach

  • Exploratory Data Analysis with visualizations showing incident distribution by call type, priority, timing, and geography
  • Unsupervised learning (clustering) to identify spatial and temporal incident patterns and hotspots
  • Regression modeling on dispatch-to-arrival response times to understand key delay factors
  • Binary classification of high-priority emergency calls for resource allocation optimization
  • Multiclass classification of incident disposition outcomes

Key Outcomes

  • Interactive Quarto-based data storytelling website with multiple sections: Home, Data Collection, Data Cleaning, EDA, Unsupervised Learning, Supervised Learning
  • Identified key factors driving response time variability across different call types and geographic areas
  • Spatial clustering revealed distinct incident hotspot patterns with actionable insights for dispatch optimization
  • Comprehensive report with visualizations designed for non-technical stakeholder communication
